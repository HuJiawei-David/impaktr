# Impaktr - Community Volunteer Tracking Platform

### Project Overview

**Impaktr** is a dual-sided web platform designed to authenticate and track community volunteer activities. This project was developed under the technical mentorship of **Jimmy [姓氏]**, focusing on translating complex business logic into scalable technical solutions.

The platform bridges the gap between organizations and volunteers, providing a verified ecosystem for social impact.
