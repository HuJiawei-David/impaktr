# 🎯 简单三步完成数据库迁移

## 您需要做的事情

**只需要 3 步，大约 2 分钟！**

---

## ⚡ 超简化版

1. **打开** https://console.neon.tech
2. **点击** "SQL Editor"
3. **粘贴下面这 4 行代码**，点击 "Run"

```sql
ALTER TYPE "NotificationType" ADD VALUE IF NOT EXISTS 'CERTIFICATE_ISSUED';
ALTER TYPE "NotificationType" ADD VALUE IF NOT EXISTS 'RANK_UP';
ALTER TYPE "NotificationType" ADD VALUE IF NOT EXISTS 'EVENT_JOINED';
ALTER TYPE "NotificationType" ADD VALUE IF NOT EXISTS 'MONTHLY_REPORT';
```

**完成！** ✅

---

## 📋 详细版（如果上面的不清楚）

### 步骤 1：登录 Neon Console

```
1. 在浏览器打开：https://console.neon.tech
2. 点击 "Sign In" 或 "Log In"
3. 使用您的账户登录（GitHub/Google/Email）
```

### 步骤 2：找到 SQL Editor

```
1. 选择您的数据库项目（通常是 "impaktr" 或类似的名字）
2. 在左侧菜单找到 "SQL Editor"
3. 点击进入
```

### 步骤 3：执行 SQL

```
1. 在编辑框中粘贴上面的 4 行 SQL 代码
2. 点击右上角的 "Run" 按钮
3. 等待几秒钟
4. 看到 "Success" 或绿色对勾 ✓
```

## 🤔 如果遇到问题？

### 问题：找不到项目
- 检查 Vercel 的环境变量，找到 DATABASE_URL
- 或者在 Neon 中创建一个新项目

### 问题：SQL 报错
- 确保复制了**完整的 4 行**代码
- 确保没有多余的空格
- 尝试一行一行执行

### 问题：不知道是不是对的数据库
- 在 Vercel 中查看 DATABASE_URL
- 应该包含 neon.tech 的地址
